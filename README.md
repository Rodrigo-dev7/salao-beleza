
﻿# Studio Aurora | Salao de Beleza

Site institucional mobile-first para um salao de beleza pequeno, com visual moderno, elegante e responsivo.

O projeto foi criado em HTML, CSS e JavaScript puros, com foco em carregamento rapido, boa leitura no celular e uma estetica sofisticada inspirada em tons escuros, champagne e rose.

## Visao Geral

O site apresenta o Studio Aurora como um espaco acolhedor para cuidados de beleza, com destaque para agendamento via WhatsApp e uma experiencia pensada primeiro para dispositivos moveis.

Principais secoes:

- Inicio com hero visual, frase de impacto e chamada para WhatsApp.
- Sobre com apresentacao acolhedora do salao.
- Servicos em cards elegantes.
- Contato com WhatsApp, e-mail e localizacao.

## Tecnologias

- HTML5 semantico
- CSS3 mobile-first
- JavaScript vanilla
- Asset local gerado para o hero
- Sem frameworks ou dependencias externas obrigatorias

## Como Visualizar

Abra o arquivo `index.html` diretamente no navegador.

O site tambem pode ser publicado facilmente em GitHub Pages, Vercel, Netlify ou qualquer hospedagem estatica.

## Personalizacao

Os dados principais podem ser alterados diretamente no `index.html`:

- Nome do salao: `Studio Aurora`
- WhatsApp: `5511999999999`
- E-mail: `contato@studioaurora.com.br`
- Endereco: `Rua das Flores, 123 - Centro`
- Textos das secoes e servicos

As cores, fontes, sombras, espacamentos e bordas estao concentrados nas variaveis do arquivo `styles.css`.

## Recursos Implementados

- Layout responsivo para celular, tablet e desktop
- Menu mobile acessivel
- Botoes grandes e faceis de tocar
- Rolagem suave entre secoes
- Animacoes leves com `IntersectionObserver`
- Respeito a `prefers-reduced-motion`
- Ano automatico no rodape
- Links prontos para WhatsApp, e-mail e Google Maps
